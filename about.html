About Daruka VLab

An educational virtual laboratory for autonomous firefighting robotics.

Project Overview

Daruka VLab is an interactive virtual laboratory designed to teach the principles of autonomous firefighting robots. Through hands-on experiments, students explore real-time fire detection using YOLOv8, autonomous navigation with obstacle avoidance, and complete finite state machine (FSM) control logic.

The project follows the Glass-Box AI philosophy — making AI decision-making transparent and understandable by exposing confidence scores, bounding boxes, and state transitions to the learner.

Lab Experiments

Experiment 1: Real-time fire detection using webcam + YOLOv8 with adjustable confidence thresholds.

Experiment 2: Autonomous navigation simulation with configurable obstacles, terrain, and speed.

Experiment 3: Full 7-state FSM mission simulation from detection through extinguishing and verification.

Technical Stack

Frontend: HTML5, CSS3, Vanilla JavaScript, Google Model Viewer for 3D.

Backend: Python Flask + YOLOv8 (Ultralytics) hosted on Hugging Face Spaces.

Offline-capable frontend with simulated fallback when backend is unavailable.
